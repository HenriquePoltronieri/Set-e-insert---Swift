# 🔠 Set e Insert - Swift

Atividade desenvolvida em **Swift** utilizando **Xcode Playground**, com foco no uso de `Set`, operações entre conjuntos e dicionários com valores do tipo array.

O projeto explora duas partes principais:

**1. Operações com Set:** dois conjuntos de caracteres são criados a partir do nome e sobrenome do autor (`HENRIQUE` e `POLTRONIERI`). Em seguida, são calculadas a **união** (todos os caracteres únicos dos dois conjuntos) e a **interseção** (caracteres presentes em ambos).

**2. Dicionário de linguagens:** um dicionário `[String: [String]]` associa o nome de cada pessoa a uma lista de linguagens de programação que ela conhece.

---

## 📤 Saída esperada

```
União: ["H", "E", "N", "R", "I", "Q", "U", "P", "O", "L", "T"]
Interseção: ["E", "N", "R", "I"]
Dicionario: ["Swift", "Flutter", "Java"]
```

> A ordem dos elementos pode variar, pois `Set` não é ordenado.
